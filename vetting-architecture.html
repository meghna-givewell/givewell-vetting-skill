Vetting Skill — Agent Architecture

How the /vetting skill orchestrates 44 agents across four waves to produce a fully-reviewed spreadsheet.

WAVE 1 Formula & Structure Checks  ·  14 agents, all parallel
Arithmetic & Logic (rows split into halves)
PAIR A·B — rows 1→½
Formula-Check-Arithmetic
Cell references, arithmetic correctness, label-formula alignment, scalar multipliers
PAIR C·D — rows ½→end
Formula-Check-Arithmetic
Second half of sheet; same checks as A·B
Data, Edge Cases & Structure (full sheet)
PAIR A·B
Formula-Check-Data
External data verification: GBD, trial papers, cross-model values
PAIR A·B
Formula-Check-Edge-Cases
Zero denominators, blank refs, silenced errors, aggregation gaps
PAIR A·B
Formula-Check-Structure
Missing cost/benefit components, scope mismatches, parallel scenario symmetry
PAIR A·B
Source-Data-Check
Raw-data tab plausibility: transpositions, year-over-year anomalies
PAIR A·B
Consistency-Check
Units, moral weights, cross-CEA parameters, study-derived parameter row consistency
Researcher Checkpoint
⏸

Pause & Present

Wave 1 findings flagged "Researcher Judgment Needed" are surfaced before Wave 2 launches. The researcher can clarify intent or confirm deviations. This prevents Wave 2 agents from spending effort on known non-issues.

WAVE 2 Independent Verification  ·  17 agents, all parallel
A·B Paired agents (run independently, no cross-knowledge)

WAVE 2.5 Reconciliation  ·  15 agents, all parallel
One reconciliation agent per A·B pair
Reads both A's and B's findings, re-reads divergent cells, and classifies each as: Confirmed · A-only · B-only · Won't Fix · Needs Researcher Input. Net-new findings written to each pair's 10-row overflow buffer.
WAVE 3 Final Review  ·  3 agents, sequential
1
Final-Review-Compaction
Routes misrouted findings (Adjustment Issue findings always stay in Findings), deduplicates, sorts by severity/CE-impact, assigns Finding IDs. Rewrites both Findings and Publication Readiness sheets sequentially.
2
Final-Review-Validation
Confirms proposed fixes don't break downstream cells. Checks confidence intervals. Fills in missing CE impact estimates for all High findings.
3
Final-Review-Dashboard
Writes Dashboard tab (CE direction, findings by sheet, tabs vetted). Presents Key Findings summary in chat.

DashboardOne-page summary + Key Findings in chat
