WAVE 2.5 Reconciliation  ·  16 agents, all parallel
One reconciliation agent per A·B pair — 16 pairs total
Re-reads every divergent cell in FORMULA mode; classifies as Confirmed, retained, Won't Fix, or Needs Researcher Input. Default is Retain. Flags silent failures via missing completion marker.
WAVE 3 Final Review  ·  4 agents, sequential
1
Final-Review-Compaction
Routes misrouted findings, deduplicates, normalises labels, sorts by severity → CE impact → error type, assigns Finding IDs.
2
Final-Review-Gap-Fill
Catches systematic A/B misses: cascade cells downstream of confirmed fixes; 15+ consecutive unflagged rows; Won't Fix spot-checks via Finding ID gaps.
3
Final-Review-Validation
CE baseline re-read; fix-validation (proposed formulas don't break downstream cells); CI sheet cross-reference; stale placeholder scan; CE impact completeness across High + Medium findings; hyperlink conversion for all column C cell refs.
4
Final-Review-Dashboard
Writes Dashboard (CE direction, findings by sheet, unvetted tabs). Presents Key Findings summary in chat.
